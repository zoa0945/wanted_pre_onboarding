# wanted_pre_onboarding
원티드 프리온보딩 ios 코스 선발과제입니다.

# Introduction
Openweather API를 이용한 날씨 정보 수집 및 데이터 전달

1. URLSession을 활용한 API 통신
2. Storyboard를 이용한 UI 구성 및 TableView를 이용한 데이터 표시

# Development Environment
* Swift
* Storyboard
* UIKit
